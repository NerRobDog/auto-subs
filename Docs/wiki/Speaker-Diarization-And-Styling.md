# Speaker Diarization And Styling

Source: [DeepWiki `6.4-speaker-diarization-and-styling`](https://deepwiki.com/tmoroney/auto-subs/6.4-speaker-diarization-and-styling)

## Purpose

Diarization identifies distinct speakers in audio. Styling features let users control how each speaker appears in subtitle output.

## Workflow

1. run transcription
2. run or apply diarization
3. review speaker assignments
4. edit labels and visual styles
5. export or send back to Resolve

## User Impact

Speaker-aware styling is one of the most visible quality improvements for creator workflows and timeline readability.
