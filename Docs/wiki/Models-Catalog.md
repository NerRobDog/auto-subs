# Models Catalog

Source: [DeepWiki `7.2-models-catalog`](https://deepwiki.com/tmoroney/auto-subs/7.2-models-catalog)

## Purpose

The models catalog is the source of truth for model metadata presented in the application.

## Includes

- model identifiers and display names
- model categories and capabilities
- selection and sort metadata
- download source mapping

## Usage

UI model lists, filtering controls, and backend model resolution all depend on this catalog layer staying consistent.
