# Type Definitions

Source: [DeepWiki `7.1-type-definitions`](https://deepwiki.com/tmoroney/auto-subs/7.1-type-definitions)

## Purpose

Shared TypeScript interfaces define contracts between frontend screens, API layers, and backend-driven data payloads.

## Typical Domains

- transcript records
- subtitle segment structures
- speaker metadata
- model definitions
- app configuration values

## Guidance

When changing API payloads, update related type definitions first and then propagate changes through UI and backend mappings.
