# Dual Operating Modes

Source: [DeepWiki `6.1-dual-operating-modes`](https://deepwiki.com/tmoroney/auto-subs/6.1-dual-operating-modes)

## Modes

- Standalone Mode
- DaVinci Resolve Mode

## Shared Behaviors

Both modes use the same core transcription and subtitle editing capabilities.

## Mode-Specific Behaviors

- standalone mode starts from user-selected media files
- Resolve mode starts from the integration bridge and editor context

## UX Goal

Users should move between modes without learning a different product, while still seeing mode-appropriate controls and data sources.
