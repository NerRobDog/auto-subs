# Subtitle Management

Source: [DeepWiki `5.3-subtitle-management`](https://deepwiki.com/tmoroney/auto-subs/5.3-subtitle-management)

## Purpose

Subtitle management covers how transcribed segments are displayed, edited, searched, and exported.

## Main Capabilities

- inline text editing
- segment timing and ordering controls
- search and filtering behaviors
- speaker-assigned segment presentation
- export paths (for example SRT and related outputs)

## User Value

This is where raw transcription becomes usable output for creators and editors.
