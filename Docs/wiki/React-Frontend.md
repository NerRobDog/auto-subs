# React Frontend

Source: [DeepWiki `3.4-react-frontend`](https://deepwiki.com/tmoroney/auto-subs/3.4-react-frontend)

## Purpose

The frontend provides all user-facing workflows and communicates with the backend through Tauri commands and local integration APIs.

## Main Structure

- `AutoSubs-App/src/pages/` for screen-level flows
- `AutoSubs-App/src/components/` for reusable UI
- `AutoSubs-App/src/contexts/` for app state
- `AutoSubs-App/src/api/` for integration clients

## Core Responsibilities

- transcription workspace and controls
- subtitle and speaker editing UI
- settings and configuration
- model management UX
- release and update presentation states
