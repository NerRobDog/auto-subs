# Transcription Workspace

Source: [DeepWiki `5.2-transcription-workspace`](https://deepwiki.com/tmoroney/auto-subs/5.2-transcription-workspace)

## Purpose

The transcription workspace is the primary interaction area where users configure jobs, run transcription, and monitor progress.

## Core Responsibilities

- input selection
- model and language settings
- workflow options (standalone and Resolve-related)
- job execution controls
- progress and status display

## Related Features

The workspace is tightly connected to model management, diarization options, and subtitle post-processing views.
