# Setup And Prerequisites

Source: [DeepWiki `9.1-setup-and-prerequisites`](https://deepwiki.com/tmoroney/auto-subs/9.1-setup-and-prerequisites)

## Required Tooling

- Node.js 18+
- npm
- Rust toolchain
- Tauri CLI 2.x

## Repository Setup

```bash
cd AutoSubs-App
npm install
```

## Start Development App

```bash
npm run tauri dev
```

## Optional Requirements

Platform-specific packaging and signing flows require additional OS-specific tools and secrets beyond the local dev baseline.
