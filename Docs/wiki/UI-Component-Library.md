# UI Component Library

Source: [DeepWiki `5.5-ui-component-library`](https://deepwiki.com/tmoroney/auto-subs/5.5-ui-component-library)

## Purpose

Shared UI primitives in `AutoSubs-App/src/components/ui/` provide consistent behavior and styling across pages.

## Characteristics

- reusable wrappers around base UI primitives
- shared styling conventions
- common interaction states and accessibility behavior
- reduced duplication in page-level components

## Practical Rule

When adding new UI patterns, prefer extending shared primitives before introducing ad-hoc one-off components.
