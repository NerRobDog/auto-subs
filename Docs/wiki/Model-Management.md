# Model Management

Source: [DeepWiki `6.3-model-management`](https://deepwiki.com/tmoroney/auto-subs/6.3-model-management)

## Purpose

Model management controls discovery, download, storage, and selection of transcription models.

## Core Responsibilities

- model catalog display in UI
- download and cache lifecycle
- installed-model status tracking
- deletion and cleanup behavior
- compatibility with platform-specific runtime constraints

## Important For This Fork

If release channels diverge, ensure model metadata and default selections remain explicit so users do not get different behavior without clear UI signals.
