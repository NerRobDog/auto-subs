# File Formats

Source: [DeepWiki `7.3-file-formats`](https://deepwiki.com/tmoroney/auto-subs/7.3-file-formats)

## Purpose

This page covers runtime file formats handled by AutoSubs.

## Main Format Areas

- transcript JSON and internal state files
- subtitle export formats such as SRT
- naming conventions for generated outputs
- import parsing behavior where supported

## Practical Note

Format compatibility is user-facing and hard to change later, so migrations and backward compatibility should be explicit.
